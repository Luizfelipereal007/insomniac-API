# insomniac-API
Este bot tem como objetivo impedir que a API entre em modo de hibernação. Ele envia requisições automáticas a cada 10 minutos, somente no período das 06:00 às 18:00 (horário comercial), garantindo que a API permaneça ativa e pronta para uso.  Fora desse intervalo, o bot permanece inativo para evitar consumo desnecessário de recursos.
